# Text editor

This app uses React-Vite and Vitest for unit test framework

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/guide/)
- [Vitest](https://vitest.dev/)

## Getting Started

Requirements:

- Node 18+

Install dependencies

```
npm install
```

Run dev environment

```
npm run dev
```

Run test

```
npm run test
```
